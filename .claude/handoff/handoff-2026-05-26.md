## 決定事項

### Phase 3 / Phase 4 は保留（旦那様意思）
2026-05-26 SDK 調査で確定:
- Even Hub SDK 0.0.10 公開 API は audioControl / imuControl / getDeviceInfo / getUserInfo / setLocalStorage の 5 つのみ
- ❌ GPS Background / HealthKit / WatchConnectivity / 心拍 API すべて存在しない
- ❌ 公式 `/background-state` skill は state snapshot 用で、JS 実行継続 (= GPS watcher 継続) ではない
- 公式 docs に明文: "No direct Bluetooth access, no arbitrary pixel drawing, no audio output, ... no camera, and images are greyscale only."
- **旦那様の意思で Even Realities への feature request も見送り**

### iPhone UI は Even OS 2.0 公式準拠
- 公式は **ライトテーマ**（私が当初 dark で実装したのは公式と正反対）
- フォントは FK Grotesk、代替で Inter / Helvetica Neue
- 緑は **デバイス接続状態にのみ使用**（公式 TC-Green ルール）

### Storage 設計
- StoragePort interface + Hub adapter + Browser adapter の port 注入パターン
- Hub 接続成功時に SDK storage に差し替え、それまで browser localStorage で動作

### 履歴保存トリガー
- `reset()` で `distanceM > 0` かつ status !== running なら自動保存
- mode は reset 後も維持（次のラン用）

### R1 リング判別
- SDK 型上、`Sys_ItemEvent` には `eventSource` あり、`Text_ItemEvent` には**無し**
- 実機で sysEvent / textEvent どちらに来るか不明 → `console.debug` で確認後、v0.4.1 で確定実装
- 現状は textEvent SCROLL も保守的に R1 由来として処理
